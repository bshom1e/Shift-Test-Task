# Запуск

## Через util.jar из папки с проектом
java -jar util.jar [опции] файл1 файл2 ...

#### Пример:
java -jar util.jar -s -o results -p filtered_ input1.txt input2.txt

## Сборка
#### 1. Клонируем репозиторий (или переходим в папку проекта)
cd shift_test
#### 2. Собираем проект (создаст util.jar в папке target)
mvn clean package
#### 3. Запускаем собранный JAR
java -jar target/util.jar [опции] файл1 файл2 ...

## Опции командной строки:
-o <путь> - Путь для выходных файлов  
-p <префикс> - Префикс для имен выходных файлов  
-a - Режим добавления в существующие файл  
-s - Краткая статистика (только количество)  
-f - Полная статистика (мин/макс/сумма/среднее)

## Прочее
- Выходные файлы создаются только если есть соответствующий тип данных
- Программа продолжает работу при ошибках в отдельных файлах/строках
- Если пользователь ничего не ввел, выводится меню помощи
